🔭 Project Overview

Classification of celestial objects from the Sloan Digital Sky Survey (SDSS) dataset using Random Forest Classifier.

The model achieves ~99% accuracy because astronomical features (redshift and spectral bands) naturally separate different object types.

⚡ Key Insight

High accuracy is expected in astronomical classification due to clear separation of features like redshift and photometric bands (u,g,r,i,z) between stars, galaxies, and quasars.
